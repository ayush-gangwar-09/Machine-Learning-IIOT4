# Machine Learning Projects

This repository contains my Machine Learning practice tasks and mini projects implemented using Python. The goal of this repository is to practice different Machine Learning algorithms and apply them to real-world prediction problems.

## Projects Included

1. **House Price Prediction**
   - Predicts house prices using Linear Regression.
   - Uses dataset features like area, bedrooms, location etc.

2. **Accident Prediction**
   - A machine learning model to analyze and predict accident related patterns.


## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook

## Purpose

This repository is created for learning and practicing Machine Learning concepts such as:
- Data preprocessing
- Model training
- Prediction
- Data analysis
